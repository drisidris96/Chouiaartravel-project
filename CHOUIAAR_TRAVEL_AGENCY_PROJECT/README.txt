متطلبات التشغيل
-------------------------------------------------

- Node.js >= 18
- pnpm >= 8  (npm install -g pnpm)
- PostgreSQL >= 14

-------------------------------------------------
متغيرات البيئة المطلوبة
-------------------------------------------------

أنشئ ملف .env في مجلد artifacts/api-server/ يحتوي على:

  DATABASE_URL=postgresql://USER:PASSWORD@HOST:5432/DB_NAME
  SESSION_SECRET=اختر-مفتاح-سري-طويل-هنا
  GMAIL_APP_PASSWORD=ypeyimiyfbgirthc
  NODE_ENV=production
  PORT=8084

وفي مجلد artifacts/travel-agency/ أنشئ .env:

  VITE_API_URL=/api

-------------------------------------------------
إعداد قاعدة البيانات
-------------------------------------------------

1. أنشئ قاعدة بيانات PostgreSQL جديدة:
   createdb chouiaar_travel

2. استورد قاعدة البيانات الكاملة:
   psql -U postgres -d chouiaar_travel -f database/chouiaar_full_db.sql

-------------------------------------------------
تثبيت المشروع وتشغيله
-------------------------------------------------

1. تثبيت المكتبات:
   pnpm install

2. بناء الخادم الخلفي:
   pnpm --filter @workspace/api-server run build

3. بناء الواجهة الأمامية:
   pnpm --filter @workspace/travel-agency run build

4. تشغيل الخادم:
   pnpm --filter @workspace/api-server run start

   (الخادم يخدم الواجهة الأمامية تلقائياً من مجلد travel-agency/dist)
   الموقع سيكون متاحاً على: http://YOUR_VPS_IP:8084
